# 互動戰況地圖 SOP

## 對 Claude 說這段話就能開始：

---

> 我要製作一張新的互動戰況地圖，請幫我在 `war-map/data/` 新增一個 JSON 檔，
> 格式參考 `war-map/data/mali_attack_20260425.json`，
> 以下是這次的報導資料：

---

## 然後提供以下資訊：

### 1. 檔名
```
格式：地點_事件_日期
範例：iran_strike_20260501
```

### 2. 標題資訊
```
中文標題：（例）伊朗反擊地圖
英文標題：（例）Iran Counter-Strike
副標題：（例）波斯灣封鎖第三週 — 美伊地面摩擦升溫
日期：（例）2026年5月1日
狀態：breaking（突發）／ ongoing（持續中）／ resolved（已結束）
來源：（例）Reuters、AFP
```

### 3. 地圖視野
```
中心點：（例）伊朗 → 大約 32°N, 53°E
縮放層級：5（全國）/ 7（區域）/ 10（城市）
```

### 4. 地點清單（每個地點填以下欄位）

| 欄位 | 說明 | 可填值 |
|------|------|--------|
| 名稱 | 地點全名 | 中英文皆可 |
| 標籤名稱 | 地圖上顯示的短標籤 | 自由填寫 |
| 類型 | 標記顏色與樣式 | 見下表 |
| 說明 | 彈出視窗的描述文字 | 自由填寫 |
| 來源 | 資料來源 | 例：Reuters 2026.05.01 |

**標記類型對照：**

| 類型 | 顏色 | 動畫 | 使用時機 |
|------|------|------|----------|
| `confirmed` | 🔴 紅 | 脈動 | 確認發生攻擊／衝突 |
| `possible` | 🟠 橙 | 脈動 | 傳聞、待確認 |
| `capital` | 🟡 黃 | 無 | 首都、重要政治中心 |
| `reference` | 🟢 綠 | 無 | 歷史參考、背景地點 |

### 5. 路線／軸線（可選）
```
如果有需要畫出公路、攻擊路線或戰略走廊，告訴 Claude：
- 起點和終點城市名稱
- 路線顏色（紅色＝攻擊軸線 / 橙色＝公路走廊）
- 路線說明文字
```

### 6. 左上角說明文字
```
用 1-2 段話說明事件背景，Claude 會放到地圖左上角說明面板。
```
